# لوحة تحكم سمو النبلاء - Dashboard System

نظام إدارة شامل لشركة سمو النبلاء للمقاولات العامة المحدودة يتضمن واجهة أمامية وواجهة خلفية (Backend) باستخدام FastAPI.

## المميزات

- ✅ إدارة المشاريع (إضافة، تعديل، حذف، عرض)
- ✅ إدارة الخدمات
- ✅ إدارة الرسائل الواردة من نموذج الاتصال
- ✅ إدارة النشرة البريدية والمشتركين
- ✅ إدارة الإحصائيات (مشاريع منجزة، وحدات سكنية، إلخ)
- ✅ إعدادات الموقع
- ✅ لوحة تحكم تفاعلية مع إحصائيات مباشرة

## المتطلبات

- Python 3.8 أو أحدث
- pip (مدير حزم Python)

## التثبيت والإعداد

### 1. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 2. تشغيل الـ Backend

انتقل إلى مجلد backend وقم بتشغيل الخادم:

```bash
cd backend
python main.py
```

أو باستخدام uvicorn مباشرة:

```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

سيتم تشغيل الـ API على: `http://localhost:8000`

### 3. فتح الواجهة الأمامية

افتح ملف `dashboard.html` في المتصفح للوصول إلى لوحة التحكم.

أو افتح `index.html` للوصول إلى الموقع الرئيسي.

## الملفات المرفوعة

يتم حفظ الصور المرفوعة في مجلد `uploads/images/` ويمكن الوصول إليها عبر:
- URL: `http://localhost:8000/uploads/images/{filename}`

## API Endpoints

### رفع الملفات
- `POST /api/upload/image` - رفع صورة (يرجع URL الصورة)
- `GET /api/images` - الحصول على قائمة الصور المرفوعة
- `DELETE /api/images/{filename}` - حذف صورة

### المشاريع
- `GET /api/projects` - الحصول على جميع المشاريع
- `GET /api/projects/{id}` - الحصول على مشروع محدد
- `POST /api/projects` - إنشاء مشروع جديد
- `PUT /api/projects/{id}` - تحديث مشروع
- `DELETE /api/projects/{id}` - حذف مشروع

### الخدمات
- `GET /api/services` - الحصول على جميع الخدمات
- `GET /api/services/{id}` - الحصول على خدمة محددة
- `POST /api/services` - إنشاء خدمة جديدة
- `PUT /api/services/{id}` - تحديث خدمة
- `DELETE /api/services/{id}` - حذف خدمة

### الرسائل
- `GET /api/messages` - الحصول على جميع الرسائل
- `GET /api/messages/{id}` - الحصول على رسالة محددة
- `POST /api/messages` - إنشاء رسالة جديدة (من نموذج الاتصال)
- `PUT /api/messages/{id}/read` - تحديد الرسالة كمقروءة
- `DELETE /api/messages/{id}` - حذف رسالة

### النشرة البريدية
- `GET /api/newsletter` - الحصول على جميع المشتركين
- `POST /api/newsletter` - الاشتراك في النشرة البريدية
- `DELETE /api/newsletter/{id}` - إلغاء الاشتراك

### الإحصائيات
- `GET /api/stats` - الحصول على الإحصائيات
- `PUT /api/stats` - تحديث الإحصائيات

### الإعدادات
- `GET /api/settings` - الحصول على الإعدادات
- `PUT /api/settings` - تحديث الإعدادات

### لوحة التحكم
- `GET /api/dashboard/stats` - الحصول على إحصائيات لوحة التحكم

## قاعدة البيانات

يتم استخدام SQLite كقاعدة بيانات محلية. يتم إنشاء ملف `dashboard.db` تلقائياً عند أول تشغيل للخادم.

## رفع الصور

### في لوحة التحكم:
1. عند إضافة مشروع جديد، اضغط على زر "رفع صورة"
2. اختر الصورة من جهازك
3. سيتم رفعها تلقائياً وحفظها في `uploads/images/`
4. سيتم استخدام رابط الصورة في المشروع تلقائياً

### في الموقع الرئيسي:
- المشاريع والخدمات يتم تحميلها تلقائياً من الـ API
- الصور المرفوعة تظهر تلقائياً في الموقع

## ملاحظات مهمة

1. **CORS**: تم تفعيل CORS لجميع النطاقات (`allow_origins=["*"]`). في الإنتاج، يجب تحديد النطاقات المسموحة فقط.

2. **الأمان**: هذا المشروع مخصص للاستخدام المحلي/التطوير. للإنتاج، يجب إضافة:
   - نظام مصادقة (Authentication)
   - تشفير كلمات المرور
   - حماية من SQL Injection (موجودة بالفعل في SQLAlchemy)
   - Rate Limiting

3. **البيانات الافتراضية**: عند أول تشغيل، يتم إنشاء بيانات افتراضية للإحصائيات والإعدادات.
